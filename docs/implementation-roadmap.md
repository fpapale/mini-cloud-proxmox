# Implementation Roadmap

* **Phase 0:** Proxmox base prep (Networking, Storage pools, OS templates).
* **Phase 1:** K3s bootstrap (VM + CT deployment, K3s installation).
* **Phase 2:** Dashboard + Observability (Headlamp, Prometheus stack, Loki).
* **Phase 3:** GitOps (Argo CD installation and App-of-Apps setup).
* **Phase 4:** Useful apps (PostgreSQL, Redis, Nextcloud, n8n).
* **Phase 5:** Hardening, backups, optional AI, optional storage expansion (Longhorn).
